=======================================================
     SIMULADOR DE INTERCAMBIO DE CARAMELOS
=======================================================

DESCRIPCIÓN:
Aplicación web educativa desarrollada en Flask que simula
un juego de intercambio de caramelos con dos fases distintas.
Diseñada para enseñar conceptos de negociación, trabajo en 
equipo y estrategia.

ARCHIVOS PRINCIPALES:
- app_simple.py         : Aplicación principal sin base de datos
- main.py              : Punto de entrada para ejecutar
- requeriments.txt     : Solo requiere Flask
- templates/           : Plantillas HTML educativas
- reporte_simulador_caramelos.tex : Reporte técnico completo

INSTALACIÓN Y USO:
1. Instalar dependencia: pip install -r requeriments.txt
2. Ejecutar: python main.py
3. Abrir navegador en: http://localhost:5000

REGLAS DEL JUEGO:
Fase 1: 1 caramelo (🍋🥚🍐) = 1 chupetín (🍭)
Fase 2: 2🍋 + 2🥚 + 2🍐 = 1 chupetín (🍭)

CARACTERÍSTICAS:
✓ Sin base de datos (almacenamiento en memoria)
✓ Interfaz educativa con guías paso a paso
✓ Máximo 20 jugadores
✓ División automática en grupos para Fase 2
✓ Marcador en tiempo real
✓ Optimizado para Visual Studio Code
